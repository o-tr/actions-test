# ImageSlide
[ImageDeviceController](/docs/Packages/ImageDeviceController/) の管理下に追加し、他の端末へ画像を投影することを想定しています

## 設定項目
### Device Name
他の端末などに表示される端末名です  

### Target Devices
同じ `ImageDeviceController` に管理されている端末の内、投影対象にしたい端末を選択します
これは初期値であり、ゲーム内でもあとから変更することができます

### Slide Urls
あらかじめ読み込むURLを指定することができます

### Splash Image
初期状態で表示する画像を選択します
