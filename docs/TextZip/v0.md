# TextZip v0
v0.0.2 以降のすべての `ImageDeviceController` で読み込むことができるフォーマットです

## 定義
```typescript
type Rect = {
  width: number;
  height: number;
}

type ManifestItem = {
  path: string;
  rect: Rect
}

type Manifest = ManifestItem[];
```

## サンプルデータ
```json
[
  {
    "path": "0.rawimage",
    "rect": {
      "width": 612,
      "height": 792
    }
  },
  {
    "path": "1.rawimage",
    "rect": {
      "width": 612,
      "height": 792
    }
  }
]
```
